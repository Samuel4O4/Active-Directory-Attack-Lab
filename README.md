# Active Directory Penetration Testing Lab

## Overview

This repository documents a complete Active Directory penetration testing engagement performed in a controlled lab environment.

Topics include:

- Initial Enumeration
- SMB Enumeration
- LDAP Enumeration
- Kerberos Attacks
- Password Attacks
- AS-REP Roasting
- Kerberoasting
- Lateral Movement
- Privilege Escalation
- Domain Enumeration
- Credential Dumping
- Persistence
